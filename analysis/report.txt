================================================================================
     THE INTERRUPT EFFECT - Experimental Results
     Does interrupting LLM code generation change the output?
================================================================================

METHODOLOGY:
  - Same prompt used for all runs (async task scheduler with tests)
  - 3 conditions per model, 5 runs each = 30 total runs
  - COMPLETE:    Full uninterrupted generation
  - INTERRUPTED: Generation killed at 20s, then fresh re-generation
  - FRAGMENT:    Complete run cut at 50%, fragment fed back to continue
  - Models: Claude Sonnet 4.6, Claude Opus 4.6 (via Anthropic API)
