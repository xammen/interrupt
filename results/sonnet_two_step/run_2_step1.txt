**`Task` dataclass** — all required fields plus an optional `fn: AsyncCallable` that holds the actual coroutine to execute.

**`TaskScheduler` class:**

| Feature | Implementation |
|---|---|
| Priority queue | Tasks sorted descending by `priority` within each topological wave |
| Dependency resolution | Kahn's algorithm → returns `List[List[str]]` execution groups |
| Circular dependency detection | Leftover non-zero in-degrees after Kahn's → raises `CircularDependencyError` |

| `get_execution_plan()` | Returns topologically-sorted groups without executing |
| Metrics | `SchedulerMetrics` (total wall time) + per-task `TaskMetrics` (start/end/elapsed/retries) |
