**`TaskScheduler` class features:**
- **Priority queue** — tasks within each execution group are sorted by descending priority
- **Topological sort** (Kahn's algorithm) to resolve dependency graphs into concurrent execution groups
- **`CircularDependencyError`** raised when cycles are detected
- **Concurrent execution** via `asyncio.gather` with a configurable `asyncio.Semaphore` for max parallelism
- **Exponential backoff retries** — delay = `base * 2^(attempt-1)`, up to `max_retries`
- **Observer pattern** — `on(event, listener)` / `off(event, listener)` for `on_task_start`, `on_task_complete`, `on_task_fail` (supports both sync and async listeners)
- **`get_execution_plan()`** — returns ordered groups of task IDs without executing
- **Metrics tracking** — total time, per-task duration, retry counts
- **Failed-dependency propagation** — tasks whose dependencies failed are skipped and marked FAILED
