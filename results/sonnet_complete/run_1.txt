### `TaskScheduler`
| Feature | Implementation |
|---|---|
| Priority queue | Tasks sorted descending by priority within each dependency wave |
| Dependency resolution | Kahn's algorithm (BFS topological sort) → returns `List[List[str]]` execution groups |
| Circular dependency | Detected when visited count < task count after Kahn's; raises `CircularDependencyError` |
| Concurrent execution | `asyncio.gather` per wave, gated by `asyncio.Semaphore(max_concurrent)` |
| Exponential backoff | Retry *n* sleeps `base_backoff * 2^(n-1)` seconds |
| Observer pattern | `EventBus.subscribe(event, async_handler)` → fires `on_task_start`, `on_task_complete`, `on_task_fail` |
| `get_execution_plan()` | Returns priority-sorted dependency waves |
